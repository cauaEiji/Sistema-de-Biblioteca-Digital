# Sistema de Biblioteca

Sistema de gerenciamento de biblioteca desenvolvido em Python para a disciplina de Fundamentos da Programação - UTFPR.

## Descrição

Este sistema permite gerenciar uma biblioteca de forma simples e eficiente, oferecendo funcionalidades de cadastro, listagem, empréstimo, devolução e busca de livros.

## Funcionalidades

- **Cadastrar livro**: Adiciona novos livros ao acervo da biblioteca
- **Listar livros**: Exibe todos os livros cadastrados
- **Emprestar livro**: Registra o empréstimo de livros
- **Registrar devolução**: Registra a devolução de livros emprestados
- **Buscar livro**: Busca livros no sistema por diferentes critérios

## Como Executar

1. Certifique-se de ter o Python instalado em seu sistema (Python 3.x recomendado)

2. Clone ou baixe este repositório

3. Navegue até o diretório do projeto:
```bash
cd Sistema-Biblioteca
```

4. Execute o programa:
```bash
python main.py
```
